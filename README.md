# Kismet

Kismet is a social dating app where friends can bet on matches. Built primarily with Java and Spring Boot (using Maven), it aims to make matchmaking fun by letting users bet on their friends' matches.

## Features

- Place bets on friends' matches
- Social interaction around dating
- Built with Spring and Maven

## Getting Started

To run or develop Kismet:

1. Clone this repository:
   ```bash
   git clone https://github.com/nathandekeyrel/kismet.git
   ```
2. Open the project in your preferred Java IDE.
3. Build and run with Maven:
   ```bash
   mvn spring-boot:run
   ```

## Tech Stack

- Java
- Spring Boot
- Maven
- HTML
